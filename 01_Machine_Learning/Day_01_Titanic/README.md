# Day 1: Titanic Exploratory Data Analysis (EDA) 🚢

This project is part of my **21 Days of AI** challenge, where I am building 21 continuous projects progressing through Machine Learning, Deep Learning, and Generative AI. 

For Day 1, I performed a comprehensive Exploratory Data Analysis on the classic Titanic dataset to uncover structural survival patterns, handle missing data, and prepare the dataset for future predictive modeling.

## 📊 Interactive Data Report
I utilized `ydata-profiling` to generate a complete, interactive HTML report analyzing the correlations, missing values, and distributions across the dataset.

👉 **[Click here to access the Interactive Profiling Report](./EDA_Titanic.html)** *(Note: Since GitHub displays HTML as raw code, please download the file to your local machine to view the fully interactive widgets and charts!)*

## 🔍 Key Insights Discovered
* **Women and children had a significantly higher survival rate across all classes.**
* **Passengers in 1st class were more likely to survive than those in 3rd class.**
* **The 'Age' and 'Cabin' columns contained the highest percentage of missing values, requiring targeted imputation strategies.**

## 🛠️ Tools & Libraries Used
* **Python**
* **Pandas & NumPy** (Data manipulation and cleaning)
* **Matplotlib & Seaborn** (Data visualization)
* **ydata-profiling** (Automated deep-dive EDA)
